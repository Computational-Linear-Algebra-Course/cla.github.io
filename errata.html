Errata¶

Here we’ll list all the updates to the exercises and coursework Classroom repositories for Autumn 2020. An up to date version is in the public repository.

  1. Exercise 1.16. In the description of the exercise, it was intended that \(C\) should be stored in the upper triangular part of \(\hat{A}\), but the lower triangular part was specified instead. This has been fixed now in the notes.

  2. Exercise 2.3. There was an error in the parameterisation of the test, which is fixed in the public repository. The docstring has also been updated (dimensions of Q were transposed).

  3. Exercise 2.4. Updated the text to indicate that a copy should be made. Replaced GS_modified \(\to\) cla_utils.GS_modified in test_exercises2.py.

  4. Exercise 2.7. Don’t use out== in GS_modified_R as this relies on structure in the numpy array that may not be the same. test_GS_modified_R has been updated in test/test_exercises2.py (there were previously _L where there shouldn’t have been, due to failing to merge a PR fixing various things). Make a copy of ‘A’ before using, and loop on range(n). Finally, we have updated the parameterised \(m\) and \(n\) values to smaller ones, as this construction for \(R\) is numerically unstable.

  5. Exercise 2.10. Added a .conj() to the orthonormality test. Changed \(n\) to \(m\) in testing orthogonality for \(Q\). And edited the docstring to clarify that it is the full QR.

  6. Exercises 2, made copies to allow comparison with in-place work.

  7. Exercise 2.11, the test was always passing because of misplaced parenthesis. Test now updated.

  8. Exercise 3.26. Taking the upper triangular part of a random matrix makes a very badly conditioned matrix that is hard to compare with the result of linalg.solve. Instead, compute the Q, R factorisation of a random matrix - the test was updated.

  9. Exercise 3.20, fixed a sign.

  10. Exercise 4.1. Clarified the docstring and fixed the corresponding test.

  11. Exercise 4.4. Fixed the tests for solve_L and solve_U.

  12. Exercise 4.8. There was a spurious ] in the test.

  13. Exercise 4.9 and 4.10. Use linalg.norm in the test, and make a copy in test_solve_LUP and test_det_LUP, and use linalg.det (updated 4th Dec 2020).

  14. Exercise 4.13. I modified the example so it actually does what I wanted it to do.

  15. Exercise 5.9. I removed the last test condition as it isn’t correct.

  16. Exercise 5.10. I updated one of the test conditions.

  17. Exercise 5.11. I updated one of the test conditions (updated 4th Dec 2020).

  18. Exercise 5.12. Needed to get m in hessenberg_ev, and fixed the assertion. I fixed the test to properly test for eigenvectors.

  19. Exercise 5.15. I changed the description of termination criteria, changed the arguments, and updated the test.

  20. Exercise 5.17. I updated the test.
