cla_utils.exercises2.GS_modified_get_R(A, k)[source]¶

Given an mxn matrix A, with columns of A[:, 0:k] assumed orthonormal, return upper triangular nxn matrix R such that Ahat = A*R has the properties that 1) Ahat[:, 0:k] = A[:, 0:k], 2) A[:, k] is orthogonal to the columns of A[:, 0:k].

Inputs:

Parameters

  • A – mxn numpy array

  • k – integer indicating the column that R should orthogonalise

Outputs:

Parameters

R – nxn numpy array
